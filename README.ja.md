## NorthPin オフラインコンパスとは?

**NorthPin オフラインコンパス**は [lapnito.cz s.r.o.](https://lapnito.cz) が開発した無料のデジタルコンパスアプリで、スマートフォンの磁力計センサーを使って正確な方位を提供します — 完全オフライン、広告、解析、バックグラウンド追跡は一切ありません。磁北を表示し、磁気偏角補正により真北にも対応、磁気干渉を検出するため、読み取りが信頼できるかどうか常に把握できます。

[Google Play](https://play.google.com/store/apps/details?id=com.tomas.northpin_compass) では **NorthPin Offline Compass**、[App Store](https://apps.apple.com/us/app/northpin-true-north-compass/id6759560694) では **NorthPin True North Compass** という名前で提供されています。

ハイキング、山間部のナビゲーション、衛星アンテナの方向合わせ、あるいは自宅の窓がどの方角を向いているかを確認したいときでも、NorthPin はインターネット接続や位置情報の共有なしに、クリアで安定した方位表示を提供します。

## 主な機能

### コンパスと方位
- **安定したデジタルコンパス** — 0〜360°のスムーズな方位表示、ジャンプや揺れなし
- **磁北方位** — デバイスの磁力計センサーによるリアルタイム方位
- **真北モード** — 一度だけの位置取得または手動座標で磁気偏角を補正
- **方位表示** — 度数と方位名（N、NE、E、SE など）を表示

### 磁場分析
- **干渉検出器** — 近くの磁石、金属面、車載ホルダーがコンパス読み取りを歪めているとき警告
- **μT（マイクロテスラ）メーター** — リアルタイムで磁場強度を表示し、センサー精度を評価
- **クイック精度チェック** — 現在の環境で信頼できる読み取りが可能か即座に確認

### キャリブレーションと信頼性
- **ガイド付きキャリブレーション** — 段階的な 8 の字キャリブレーション、実用的なヒント付き
- **キャリブレーションステータス表示** — 使用前に再キャリブレーションが必要か把握
- **診断レポート** — センサーデータ付きプレーンテキストレポートを生成・共有

### ナビゲーションツール
- **Pin Direction（方位ロック）** — 方位をロックし、ターゲットを指すまで左右のガイド矢印を表示
- **方位保持** — 地形を移動中も固定の方位参照を維持
- **アウトドアモード** — 強い日光下での使用に最適化された表示設定

### ユーザー体験
- **クリーンでミニマルなインターフェース** — 屋外で読みやすいダークテーマ
- **100% オフライン動作** — インターネット接続は一切不要
- **広告なし、追跡なし** — データ収集ゼロ、バックグラウンドサービスゼロ
- **軽量** — バッテリーとストレージの消費最小、センサーはアプリ使用中のみ動作

## デジタルコンパスの仕組み

NorthPin のようなデジタルコンパスアプリは、スマートフォンに内蔵されている**磁力計センサー**からデータを読み取ります。磁力計は地球の磁場を 3 軸（X、Y、Z）で測定し、デバイスの向きに対する磁北の方向を計算します。NorthPin は生のセンサーデータを平滑化アルゴリズムで処理し、小さな動きごとにチラつかない安定した方位を生成します。

**磁気偏角とは?** 地球の磁北極は地理的（真）北と同じではありません。両者の角度差を磁気偏角と呼び、地球上のどこにいるかで変化します。NorthPin は一度だけ位置情報を取得してローカルの偏角を計算することで補正できます。位置情報を共有したくない場合は手動で座標を入力することもできます。

**屋内でコンパスが不正確に見えるのはなぜ?** 電子機器、金属家具、スピーカー、または建物の鋼構造からの磁気干渉は磁力計の読み取りを歪めます。NorthPin の μT メーターと干渉検出器はこれらの状況の識別に役立ちます。磁場強度が異常に高いか低い場合、アプリが警告します。干渉源から離れるか 8 の字キャリブレーションを行うと精度が回復します。

**マイクロテスラ（μT）とは?** マイクロテスラは磁場強度を測る単位です。地球の自然磁場は位置によって通常 25 μT から 65 μT の範囲です。NorthPin がこの範囲から大きく外れた値を示す場合、近くの磁気源が干渉している可能性があります。

## よくある質問

**Q: NorthPin オフラインコンパスは無料ですか?**
A: はい、Android と iOS の両方で完全に無料でダウンロード・使用できます。アプリ内課金、プレミアム版、サブスクリプションはありません。

**Q: NorthPin はインターネットなしで動きますか?**
A: はい、コンパスは 100% オフラインで動作します。磁力計はインターネット接続を必要としません。唯一のオプションのオンライン機能は真北の偏角補正のための一度きりの位置取得で、手動で座標を入力すればスキップ可能です。

**Q: NorthPin は真北と磁北のどちらを表示しますか?**
A: デフォルトでは磁北を表示します。設定で真北モードを有効にすると、位置情報に基づいて磁気偏角を補正します。これは多くの地図で使われる地理的な北です。

**Q: NorthPin の精度は?**
A: 精度はデバイスの磁力計の品質と環境によります。金属や電子機器から離れた開けた屋外では、ほとんどの最新スマートフォンで 1〜3°の精度が得られます。μT メーターと干渉検出器が読み取りの信頼性を判断するのに役立ちます。

**Q: コンパスの読み取りがおかしい/ブレる場合は?**
A: 通常は近くの物体からの磁気干渉が原因です — 磁石入りスマホケース、金属テーブル、車載ホルダー、電子機器など。開けた場所に移動してガイド付き 8 の字キャリブレーションを行ってください。μT メーターを確認し、値が 25〜65 μT から大きく外れていたら干渉があります。

**Q: Pin Direction 機能とは何ですか?**
A: Pin Direction は特定の方位（例：247°）をロックでき、その方向に正確に向くまで左右のガイド矢印を表示します。ハイキングでコンパス方位を追いかけたり、機器の方向を合わせたりするのに便利です。

**Q: Android 版と iOS 版の違いは?**
A: 両バージョンとも同じ中核機能を提供します。Android 版は "NorthPin Offline Compass"、iOS 版は "NorthPin True North Compass" という名前です。どちらも無料で広告なしです。

**Q: NorthPin はデータを収集しますか?**
A: いいえ。NorthPin はデータを一切収集しません。解析、広告トラッカー、クラッシュレポーター、バックグラウンドサービスは一切ありません。Apple の App Store プライバシーラベルも「データ収集なし」と確認しています。

**Q: どのデバイスに対応していますか?**
A: Android 7.0（Nougat）以降、iOS 13.0 以降です。デバイスには磁力計センサーが必要です（最新のスマートフォンはほとんど搭載しています）。

**Q: コンパスのキャリブレーション方法は?**
A: NorthPin のキャリブレーション画面を開き、ガイド付きの 8 の字の動きに従ってください。スマートフォンを手に持ち、空中でゆっくり 8 の字を描きます。アプリがキャリブレーション状態を表示し、センサーが正しく校正されたら通知します。
